## 3D文件格式说明

三维对象的电子表示称为 3D 文件。

https://docs.fileformat.com/zh/3d/

- `3MF`  : `xml` `免费` `完整`
    - https://3dviewer.net/
- `AMF`  : `xml`
    - https://3dviewer.net/
- `glb`  : `glTF` 的二进制编码
    - https://3dviewer.net/
- `glTF` : `ASCII` `GL传输格式` `结构` `相机` `材质` `动画` `网格`
    - https://3dviewer.net/
- `MTL`  : `obj`的材质模板库
- `OBJ`  : `几何图形`
    - https://3dviewer.net/
    - https://www.viewstl.com/


## CAD文件格式说明

CAD代表计算机辅助设计。

https://docs.fileformat.com/zh/cad/


- `STL`   : `CAD` `几何图形`
    - https://3dviewer.net/
    - https://www.viewstl.com/
- `DXF`   : `ASCII` AutoCAD 图形文件
- `DWG`   : 二进制 Autodesk AutoCAD
- `NC`    : Mastercam CNC
- `GCODE` : 控制计算机化机床和 3D 打印机
